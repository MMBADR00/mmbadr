Automated Production Line

Undergraduate Project | AIET | 2017

Home Projects and Grants

Overview

This page summarizes the Automated Production Line project supervised at AIET in 2017. It documents the main objectives, system layout, core components, and the control and automation approach used to coordinate the line.

Objectives

  • Design a functional production-line workflow from input handling to finished output.
  • Integrate sensors and actuators to automate key stages and improve repeatability.
  • Implement a control logic sequence to coordinate stations and ensure safe operation.
  • Validate performance through testing, troubleshooting, and documented results.

System Description

  • Line workflow: Material entry, processing stations, transfer, and exit/output.
  • Mechanical structure: Frames, mounts, guides, and station fixtures sized for stability.
  • Electrical panel: Power distribution, protection, wiring organization, and I/O interfacing.
  • Human interface: Basic operator controls (start, stop, reset) and status indication.

If you have photos, wiring diagrams, or a block diagram, you can embed them in the section below.

Sensors and Actuators

Typical Sensors

  • Limit switches or proximity sensors for position and station detection.
  • Optical sensors for part presence or counting.
  • Emergency stop and safety interlocks for safe operation.

Typical Actuators

  • DC motors or stepper motors for conveyors or transfer mechanisms.
  • Solenoids or pneumatic actuators for pushing, clamping, or diversion tasks.
  • Indicators and alarms for status and fault signaling.

Control Strategy

  • Sequence control: Step-by-step station logic with clear transition conditions.
  • Interlocks: Prevent conflicting actuator commands and unsafe transitions.
  • Fault handling: Stop or safe state on sensor disagreement or timeout conditions.
  • Reset behavior: Return the line to a known initial state before restarting.

You can add a state diagram image or a short table describing each step (state) and its entry and exit conditions.

Outcomes

  • Demonstrated automated station sequencing with reliable part transfer.
  • Validated sensor feedback and actuator timing through iterative testing.
  • Documented troubleshooting and improvements to increase robustness.
